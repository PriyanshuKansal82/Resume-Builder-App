# 🚀 RESUME BUILDER PRO

**RESUME BUILDER PRO** is an AI-powered full-stack MERN application that helps users create professional, ATS-friendly resumes with real-time preview, smart AI optimization, and modern resume templates.

## ✨ Features

* 🔐 Secure JWT Authentication
* 📝 Dynamic Resume Builder
* 🎨 Multiple Professional Templates
* 🤖 AI-Powered Content Suggestions
* 🖼️ Profile Photo Upload & Background Removal
* 🔗 Shareable Resume Links
* 📱 Fully Responsive Design
* 💾 Manage Multiple Resumes
* ⚡ Real-time Preview & Editing

## 🛠 Tech Stack

### Frontend

* React 19
* Redux Toolkit
* React Router DOM
* Tailwind CSS
* Axios
* Lucide React

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Multer

### AI & Services

* OpenAI API
* ImageKit CDN

## 📌 Purpose

Built for students, professionals, and job seekers to create modern resumes that stand out and perform well in ATS systems.
